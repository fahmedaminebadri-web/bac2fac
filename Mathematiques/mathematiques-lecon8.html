📘 1. Définition du logarithme népérien

Le logarithme népérien, noté ln, est la fonction réciproque de l'exponentielle. ln(a) est le nombre x tel que eˣ = a.

Définition
ln(a) = x  ⇔  eˣ = a     avec a > 0
📌

Domaine de définition : ]0, +∞[ (ln n'est défini que pour les nombres strictement positifs)

📌

ln(1) = 0   |   ln(e) = 1

📌

ln est la primitive de 1/x sur ]0, +∞[ qui s'annule en x = 1 :   ∫₁ᵃ (1/t) dt = ln(a)

💡 Exemples concrets

ln(e³) = 3   |   ln(1) = 0   |   ln(e) = 1

ln(7) ≈ 1.946  (car e^1.946 ≈ 7)

ln(0.5) = ln(1/2) = -ln(2) ≈ -0.693

📗 2. Propriétés algébriques de ln

Pour tous a > 0 et b > 0 :

Propriétés fondamentales
ln(a × b) = ln(a) + ln(b)
ln(a / b) = ln(a) - ln(b)
ln(aⁿ) = n × ln(a)     pour tout n ∈ ℤ
🔑

ln(1/a) = -ln(a)

🔑

ln(√a) = (1/2) ln(a)

🔑

ln(a) = ln(b) ⇔ a = b

💡 Exemples de simplification

ln(6) = ln(2×3) = ln(2) + ln(3)

ln(8) = ln(2³) = 3ln(2)

ln(e²) = 2

ln(e⁵ / e²) = ln(e³) = 3

⚠️ Erreurs fréquentes

❌ ln(a + b) = ln(a) + ln(b)  (FAUX — on ne peut PAS "distribuer" le ln sur une somme)

✅ ln(a × b) = ln(a) + ln(b)

❌ ln(a)² = 2ln(a)  (FAUX — ln(a)² = ln(a) × ln(a))

✅ ln(a²) = 2ln(a)

📙 3. Lien entre exp et ln

exp et ln sont réciproques l'une de l'autre :

Réciprocité
ln(eˣ) = x       pour tout x ∈ ℝ
e^(ln x) = x      pour tout x > 0

Cela signifie que leurs graphes sont symétriques par rapport à la droite y = x.

💡 Utilisation pratique

Pour résoudre eˣ = 5 :   x = ln(5)

Pour résoudre ln(x) = 3 :   x = e³

Pour calculer ln(√e) :   = ln(e^0.5) = 0.5

📕 4. Étude de la fonction ln(x)

📈

Domaine : ]0, +∞[

📈

Dérivée : ln'(x) = 1/x

📈

Monotonicité : strictement croissante sur ]0, +∞[

📈

Limites : lim ln(x) = -∞ (x→0⁺),   lim ln(x) = +∞ (x→+∞)

📈

Asymptotes : l'axe Oy (x=0) est asymptote verticale

📈

Continuité : ln est continue et dérivable sur ]0, +∞[

Croissance comparée (résultat fondamental)
lim (ln x) / x = 0   (quand x → +∞)
lim x·ln x = -∞   (quand x → 0⁺)

Dérivée de ln(u(x))

Dérivée de ln(u)
[ln(u(x))]' = u'(x) / u(x)     (valable si u(x) > 0)

💡 Exemples de dérivation

[ln(x²+1)]' = 2x/(x²+1)

[ln(2x+3)]' = 2/(2x+3)

[ln(eˣ)]' = eˣ/eˣ = 1

📒 5. Équations et inéquations logarithmiques

Équations

Résolution
ln(a) = ln(b)  ⇔  a = b   (a > 0, b > 0)
ln(a) = c  ⇔  a = eᶜ   (c ∈ ℝ, a > 0)

💡 Exemples

ln(x) = ln(7)  →  x = 7

ln(2x - 1) = 3  →  2x - 1 = e³  →  x = (e³ + 1)/2

ln(x) = -2  →  x = e⁻² ≈ 0.135

Inéquations

Comme ln est strictement croissante :

Inéquations
ln(a) < ln(b)  ⇔  a < b   |   ln(a) > c  ⇔  a > eᶜ

💡 Exemples

ln(x) > 0  →  x > 1  (car ln(1) = 0 et ln est croissante)

ln(2x) < ln(5)  →  2x < 5  →  x < 2.5

ln(x+3) ≥ 1  →  x+3 ≥ e  →  x ≥ e - 3

⚠️ Conditions obligatoires

Dans toute équation ou inéquation avec ln :

→ Chaque argument de ln doit être strictement positif

→ Toujours vérifier les conditions AVANT de résoudre !
