📘 1. Définition et notation

La fonction exponentielle, notée exp ou eˣ, est l'unique fonction f telle que :

Définition caractéristique
f'(x) = f(x)     et     f(0) = 1

Cela signifie que la dérivée de l'exponentielle est égale à elle-même ! C'est cette propriété extraordinaire qui la rend unique.

📌

e ≈ 2.71828… est le nombre d'Euler (constante irrationnelle)

📌

exp(x) = eˣ  →  notation interchangeable

📌

exp est définie sur ℝ tout entier

💡 Valeurs remarquables

e⁰ = 1   |   e¹ = e ≈ 2.718   |   e² ≈ 7.389   |   e⁻¹ ≈ 0.368

Pour tout x, eˣ > 0 (toujours positive)

📗 2. Propriétés algébriques

Pour tous réels a et b :

Propriétés fondamentales
eᵃ × eᵇ = eᵃ⁺ᵇ   |   eᵃ / eᵇ = eᵃ⁻ᵇ   |   (eᵃ)ⁿ = eᵃⁿ
🔑

e⁻ˣ = 1/eˣ  (parité)

🔑

eᵃ = eᵇ  ⇔  a = b  (injectivité)

🔑

Attention : eᵃᵇ ≠ eᵃ · eᵇ en général. On a eᵃᵇ = (eᵃ)ᵇ

💡 Simplification d'expressions

e² × e³ = e⁵   |   e⁶ / e² = e⁴   |   (e²)³ = e⁶

e⁵⁻ˣ / e² = e⁵⁻ˣ⁻² = e³⁻ˣ

eˣ × e⁻ˣ = e⁰ = 1

📙 3. Étude de la fonction exp(x)

📈

Domaine : ℝ (définie partout)

📈

Image : ]0, +∞[ (strictement positive)

📈

Dérivée : exp'(x) = exp(x) = eˣ

📈

Monotonicité : strictement croissante sur ℝ (car eˣ > 0)

📈

Limites : lim eˣ = +∞ quand x → +∞,   lim eˣ = 0 quand x → -∞

📈

Asymptotes : l'axe Ox (y=0) est asymptote horizontale en -∞

📊 Tableau de variation de la fonction exp
x −∞ 0 +∞
exp'(x) = eˣ +       +       +
exp(x) 0 ↗ 1 ↗ +∞

📖 Lecture du tableau :

• La dérivée eˣ est strictement positive sur ℝ, donc exp est strictement croissante.

• Quand x → −∞, eˣ tend vers 0 (sans jamais l'atteindre).

• En x = 0 : exp(0) = 1 (point sur la courbe).

• Quand x → +∞, eˣ tend vers +∞ (croissance très rapide).

📕 4. Équations et inéquations exponentielles

Équations

La fonction exp étant strictement croissante, on peut passer aux logarithmes :

Résolution
eᵃ = eᵇ  ⇔  a = b   |   eˣ = k  ⇔  x = ln(k)   (k > 0)

💡 Exemples

e²ˣ⁻³ = e⁵  →  2x - 3 = 5  →  x = 4

e³ˣ = 7  →  3x = ln(7)  →  x = ln(7)/3

eˣ = 0  →  impossible (eˣ > 0 toujours)

Inéquations

Comme exp est croissante :

Inéquations
eᵃ < eᵇ  ⇔  a < b   |   eᵃ > eᵇ  ⇔  a > b

💡 Exemple

e²ˣ > e³  →  2x > 3  →  x > 3/2

e⁻ˣ < 5  →  -x < ln(5)  →  x > -ln(5)

📒 5. Dérivée de exp(u(x))

Par composition avec la règle de dérivation :

Dérivée de exp(u)
[eᵘ⁽ˣ⁾]' = u'(x) × eᵘ⁽ˣ⁾

💡 Exemples

[e²ˣ]' = 2e²ˣ   |   [eˣ²]' = 2xeˣ²

[e^(3x+1)]' = 3e^(3x+1)

[e^(-x)]' = -e^(-x)   |   [e^√x]' = (1/(2√x)) · e^√x

⚠️ Erreurs fréquentes à éviter

❌ eˣ⁺ʸ = eˣ + eʸ (FAUX)

✅ eˣ⁺ʸ = eˣ × eʸ

❌ e²ˣ = 2eˣ (FAUX)

✅ e²ˣ = (eˣ)²
