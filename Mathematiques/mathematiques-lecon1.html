📘 1. Qu'est-ce qu'un nombre complexe ?

On dit que ℝ (l'ensemble des nombres réels) est insuffisant pour résoudre certaines équations. Par exemple, l'équation x² + 1 = 0 n'a aucune solution réelle car aucun nombre réel élevé au carré ne donne -1.

Pour résoudre ce problème, on introduit le nombre i, appelé unité imaginaire, tel que :

Définition fondamentale
i² = -1

Un nombre complexe est un nombre de la forme :

Forme algébrique d'un complexe
z = a + bi     avec a ∈ ℝ et b ∈ ℝ

où a est la partie réelle et b est la partie imaginaire.

✅

Si b = 0 → z est un nombre réel (ℝ ⊂ ℂ)

✅

Si a = 0 et b ≠ 0 → z est un imaginaire pur

💡 Exemple concret

z₁ = 3 + 5i  →  Re(z₁) = 3,   Im(z₁) = 5

z₂ = -2 + 4i  →  Re(z₂) = -2,   Im(z₂) = 4

z₃ = 7  →  z₃ ∈ ℝ (donc z₃ ∈ ℂ)

z₄ = 3i  →  z₄ est un imaginaire pur

📗 2. Forme algébrique et opérations

Les opérations sur les nombres complexes respectent les mêmes règles que sur les réels, avec la règle i² = -1.

➕ Addition et soustraction

Formule
(a + bi) + (c + di) = (a + c) + (b + d)i

💡 Exemple

(3 + 5i) + (-2 + 4i) = (3 - 2) + (5 + 4)i = 1 + 9i

(3 + 5i) - (-2 + 4i) = (3 + 2) + (5 - 4)i = 5 + i

✖️ Multiplication

Formule
(a + bi)(c + di) = (ac - bd) + (ad + bc)i

💡 Exemple

(2 + 3i)(1 - 4i) = 2×1 + 2×(-4i) + 3i×1 + 3i×(-4i)

= 2 - 8i + 3i - 12i²  →  = 2 - 5i - 12(-1) = 2 - 5i + 12 = 14 - 5i

➗ Division

Pour diviser deux complexes, on multiplie le numérateur et le dénominateur par le conjugué du dénominateur.

Formule
(a + bi) / (c + di) = (a + bi)(c - di) / (c² + d²)

💡 Exemple

(1 + 2i) / (3 - i)  = (1 + 2i)(3 + i) / (3² + (-1)²)

= (3 + i + 6i + 2i²) / 10 = (3 + 7i - 2) / 10 = (1 + 7i) / 10 = 0.1 + 0.7i

📙 3. Conjugué et module

Le conjugué

Le conjugué de z = a + bi est :

Conjugué
z̄ = a - bi

💡 Exemples

z = 3 + 5i  →  z̄ = 3 - 5i

z = -2 + 4i  →  z̄ = -2 - 4i

Propriété : z × z̄ = a² + b²  (toujours réel positif ou nul)

Le module

Le module de z = a + bi est sa "distance à l'origine" :

Module
|z| = √(a² + b²)
📐

|z| ≥ 0 toujours, et |z| = 0  ⇔  z = 0

📐

|z|² = z × z̄ = a² + b²

📐

|z₁ × z₂| = |z₁| × |z₂|

📐

|z₁ / z₂| = |z₁| / |z₂|

💡 Exemple

z = 3 + 4i  →  |z| = √(9 + 16) = √25 = 5

📕 4. Forme trigonométrique

Tout nombre complexe non nul peut s'écrire sous forme trigonométrique :

Forme trigonométrique
z = r(cos θ + i sin θ)     avec r = |z| ≥ 0 et θ = Arg(z)

où θ est l'argument de z, noté Arg(z) : c'est l'angle que fait le vecteur OM avec l'axe des réels positifs.

🔁

Arg(z) est défini à 2π près : Arg(z) = θ + 2kπ, k ∈ ℤ

🔁

Pour retrouver a et b : a = r cos θ,   b = r sin θ

💡 Exemple

z = 1 + i  →  r = √(1+1) = √2

cos θ = 1/√2 et sin θ = 1/√2  →  θ = π/4

donc : z = √2(cos(π/4) + i sin(π/4))

📒 5. Formules d'Euler et de Moivre

Formules d'Euler

Formules d'Euler
cos θ = (e^(iθ) + e^(-iθ)) / 2       sin θ = (e^(iθ) - e^(-iθ)) / 2i

Formule de Moivre

Formule de De Moivre
[cos θ + i sin θ]^n = cos(nθ) + i sin(nθ)     pour tout n ∈ ℤ

💡 Exemple

(cos(π/3) + i sin(π/3))² = cos(2π/3) + i sin(2π/3)

= (-1/2) + i(√3/2)

⚠️ Astuce de calcul rapide

Si z = r(cos θ + i sin θ) et z' = r'(cos θ' + i sin θ'), alors :

z × z' = rr' (cos(θ+θ') + i sin(θ+θ'))

zⁿ = rⁿ (cos(nθ) + i sin(nθ))

📓 6. Équations dans ℂ

Équations du second degré

Soit l'équation az² + bz + c = 0 avec a ≠ 0. Le discriminant Δ = b² - 4ac.

📊

Si Δ > 0 → deux solutions réelles

📊

Si Δ = 0 → une solution réelle double

📊

Si Δ < 0 → deux solutions complexes conjuguées

Solutions complexes
z₁ = (-b + i√|Δ|) / 2a      z₂ = (-b - i√|Δ|) / 2a

💡 Exemple

z² + 2z + 5 = 0  →  Δ = 4 - 20 = -16

√|Δ| = √16 = 4

z₁ = (-2 + 4i) / 2 = -1 + 2i

z₂ = (-2 - 4i) / 2 = -1 - 2i

⚠️ Point clé à retenir

Quand Δ < 0, les deux solutions sont toujours conjuguées l'une de l'autre (z₁ = z̄₂). C'est une propriété fondamentale des coefficients réels.
