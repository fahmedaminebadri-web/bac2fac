📘 1. Définition et modes de génération

Une suite numérique est une fonction de ℕ dans ℝ :

Notation
u : ℕ → ℝ   |   u(n) = uₙ   |   La suite est notée (uₙ)ₙ∈ℕ ou simplement (uₙ)

Deux façons de définir une suite

📝

Formule explicite : uₙ = f(n) — on peut calculer directement uₙ en fonction de n

🔄

Relation de récurrence : uₙ₊₁ = f(uₙ) — on a besoin du terme précédent pour calculer le suivant

💡 Exemples

Explicite : uₙ = 3n + 2 → u₀ = 2, u₁ = 5, u₂ = 8, u₃ = 11…

Récurrence : u₀ = 3, uₙ₊₁ = 2uₙ + 1 → u₁ = 7, u₂ = 15, u₃ = 31…

📗 2. Suite arithmétique

Une suite est arithmétique si on passe d'un terme au suivant en ajoutant toujours le même nombre (la raison r) :

Suite arithmétique
uₙ₊₁ = uₙ + r   |   uₙ = u₀ + n×r   |   uₙ = uₚ + (n-p)×r
📐

Terme général : uₙ = u₀ + n·r

📐

Somme des premiers termes : Sₙ = n × (u₀ + uₙ) / 2 = n × (2u₀ + (n-1)r) / 2

📐

Monotonicité : si r > 0 → croissante, si r < 0 → décroissante, si r = 0 → constante

💡 Exemple

Suite arithmétique avec u₀ = 5 et r = 3

uₙ = 5 + 3n  →  u₀ = 5, u₁ = 8, u₂ = 11, u₃ = 14…

S₁₀ = 10 × (5 + 35) / 2 = 10 × 40 / 2 = 200

📙 3. Suite géométrique

Une suite est géométrique si on passe d'un terme au suivant en multipliant toujours par le même nombre (la raison q) :

Suite géométrique
uₙ₊₁ = uₙ × q   |   uₙ = u₀ × qⁿ   |   uₙ = uₚ × qⁿ⁻ᵖ
📐

Terme général : uₙ = u₀ × qⁿ

📐

Somme des premiers termes : Sₙ = u₀ × (1 - qⁿ⁺¹) / (1 - q)   si q ≠ 1

📐

Monotonicité : dépend du signe et de la valeur de q et u₀

💡 Exemple

Suite géométrique avec u₀ = 2 et q = 3

uₙ = 2 × 3ⁿ  →  u₀ = 2, u₁ = 6, u₂ = 18, u₃ = 54…

S₃ = 2 × (1 - 3⁴) / (1 - 3) = 2 × (1 - 81) / (-2) = 2 × 40 = 80

⚠️ Applications concrètes des suites géométriques

• Intérêts composés : Cₙ = C₀ × (1 + t)ⁿ

• Évolution en pourcentage : une grandeur qui augmente de 5% chaque année → q = 1.05

• Évolution décroissante : une population qui diminue de 3% → q = 0.97

📕 4. Limites de suites

Une suite (uₙ) peut :

➡️

Converger vers un réel L : lim uₙ = L (les termes se rapprochent de L)

⬆️

Diverger vers +∞ : lim uₙ = +∞ (les termes deviennent aussi grands qu'on veut)

⬇️

Diverger vers -∞ : lim uₙ = -∞

↔️

Ne pas avoir de limite : suite périodique ou chaotique

Limites de référence

Limites fondamentales
lim n = +∞   |   lim n² = +∞   |   lim qⁿ = +∞ si q > 1  |  lim qⁿ = 0 si |q| < 1

Opérations sur les limites

💡 Formes indéterminées

0 × ∞   |   ∞ - ∞   |   0/0   |   ∞/∞   |   0⁰   |   1^∞

Ces formes nécessitent un traitement spécial (factorisation, théorèmes de comparaison…)

📐

Théorème des Gendarme : si uₙ ≤ vₙ ≤ wₙ et lim uₙ = lim wₙ = L → lim vₙ = L

📐

Suite bornée + convergence : toute suite convergente est bornée

📒 5. Raisonnement par récurrence

Pour démontrer qu'une propriété Pₙ est vraie pour tout n ∈ ℕ, on fait 3 étapes :

1️⃣

Initialisation : vérifier que P₀ est vraie

2️⃣

Hérédité : supposer Pₖ vraie, montrer que Pₖ₊₁ est vraie

3️⃣

Conclusion : par récurrence, Pₙ est vraie pour tout n

💡 Démontrer que uₙ = 3ⁿ + 2 est croissante

Initialisation : u₀ = 3⁰ + 2 = 3, u₁ = 3 + 2 = 5 → u₁ > u₀ ✓

Hérédité : Supposons uₖ₊₁ > uₖ

Alors 3^(k+2) + 2 = 3 × 3^(k+1) + 2 > 3 × 3^k + 2 = uₖ₊₁ + 2 > uₖ₊₁ ✓

Conclusion : la suite est croissante pour tout n.

⚠️ Types de récurrence

• Récurrence simple : on utilise Pₖ pour démontrer Pₖ₊₁

• Récurrence double : on utilise Pₖ et Pₖ₊₁ pour démontrer Pₖ₊₂

• Récurrence forte : on suppose que P₀, P₁, ..., Pₖ sont vraies pour montrer Pₖ₊₁
