📘 1. Primitive d'une fonction

Une primitive F d'une fonction f sur un intervalle I est une fonction dérivable telle que :

Définition
F'(x) = f(x)     pour tout x ∈ I
📌

Si F est une primitive de f, alors toutes les primitives sont de la forme F + C, où C est une constante

📌

Une fonction continue sur un intervalle admet toujours des primitives

Primitives usuelles

Table des primitives
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C   (n ≠ -1)
∫ 1/x dx = ln|x| + C   |   ∫ eˣ dx = eˣ + C   |   ∫ cos(x) dx = sin(x) + C
∫ sin(x) dx = -cos(x) + C   |   ∫ 1/(1+x²) dx = arctan(x) + C

💡 Exemples

Primitive de f(x) = 3x² :    F(x) = x³ + C

Primitive de f(x) = 1/x :     F(x) = ln|x| + C

Primitive de f(x) = e²ˣ :     F(x) = (1/2)e²ˣ + C

📗 2. Intégrale définie et théorème fondamental

Si F est une primitive de f sur [a, b], l'intégrale définie est :

Théorème fondamental de l'analyse
∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) - F(a)

💡 Exemple

∫₀² x² dx = [x³/3]₀² = (8/3) - 0 = 8/3

Graphiquement : c'est l'aire sous la courbe y = x² entre x = 0 et x = 2

⚠️ Intégrale et aire

Si f(x) ≥ 0 sur [a, b] : ∫ₐᵇ f(x) dx = aire entre la courbe et l'axe Ox

Si f(x) ≤ 0 : l'intégrale est négative (aire algébrique)

L'aire réelle = ∫|f(x)| dx

📙 3. Propriétés des intégrales

📐

Linéarité : ∫(af + bg) dx = a∫f dx + b∫g dx

📐

Relation de Chasles : ∫ₐᵇ f + ∫ᵇᶜ f = ∫ₐᶜ f

📐

Inversion des bornes : ∫ₐᵇ f = -∫ᵇₐ f

📐

Positivité : si f ≥ 0 sur [a,b] alors ∫ₐᵇ f ≥ 0

📐

Ordre : si f ≤ g alors ∫ₐᵇ f ≤ ∫ₐᵇ g

Valeur moyenne

Valeur moyenne
μ = (1/(b-a)) × ∫ₐᵇ f(x) dx

💡 Exemple

Valeur moyenne de f(x) = x² sur [0, 2] :

μ = (1/2) × ∫₀² x² dx = (1/2) × (8/3) = 4/3

📕 4. Intégration par parties

Quand les méthodes directes ne suffisent pas, on utilise l'intégration par parties (IPP) :

Formule d'intégration par parties
∫ u(x) · v'(x) dx = u(x)·v(x) - ∫ u'(x)·v(x) dx

En version définie :

IPP définie
∫ₐᵇ u(x)·v'(x) dx = [u(x)·v(x)]ₐᵇ - ∫ₐᵇ u'(x)·v(x) dx
🔑

Règle mnémotechnique : IPPT = Intégrer, Dériver, Permuter, Tester

🔑

À choisir en priorité pour u : logarithmes, puissances de x

🔑

À dériver (donc à choisir en second) : exponentielles, sinus, cosinus

💡 Exemple : ∫ x·eˣ dx

Posons u = x → u' = 1   |   v' = eˣ → v = eˣ

∫ x·eˣ dx = x·eˣ - ∫ eˣ dx = x·eˣ - eˣ + C = eˣ(x - 1) + C

📒 5. Applications : aires et volumes

Aire entre deux courbes

L'aire entre f et g entre a et b (avec f ≥ g) est :

Aire entre deux courbes
A = ∫ₐᵇ |f(x) - g(x)| dx

Volume d'un solide de révolution

Le volume du solide obtenu par rotation de la courbe y = f(x) autour de l'axe Ox, entre x = a et x = b :

Volume de révolution
V = π × ∫ₐᵇ [f(x)]² dx

💡 Exemple : Volume d'un cône de révolution

Un cône de rayon R et hauteur H peut être vu comme la rotation de y = (R/H)x autour de Ox.

V = π × ∫₀ᴴ (R²/H²)·x² dx = π × (R²/H²) × [x³/3]₀ᴴ

V = π × (R²/H²) × (H³/3) = (1/3)πR²H ✓

⚠️ Formules d'aires à retenir

• Disque : A = πR²

• Couronne circulaire : A = π(R² - r²)

• Trapèze : A = ((b₁ + b₂) / 2) × h
