📘 1. Qu'est-ce que Qt Designer ?

Qt Designer est un outil visuel inclus dans Qt qui permet de créer des interfaces graphiques (fenêtres, boutons, champs de texte…) sans écrire une seule ligne de code. On construit l'interface en glisser-déposer.

Une fois l'interface terminée, Qt Designer génère un fichier .ui (format XML). Ce fichier peut ensuite être utilisé par un développeur dans son programme.

🎯

But — Créer visuellement des fenêtres et des éléments d'interface

🖱️

Méthode — Glisser-déposer des éléments depuis une boîte à outils

📄

Résultat — Un fichier .ui qui décrit l'interface

✅

Avantage — Pas besoin de coder l'interface, gain de temps énorme

💡 Comment lancer Qt Designer ?

Sous Windows : cherchez "Designer" dans le menu Démarrer (après installation de PyQt5)

Ou dans le terminal : tapez designer

⚠️ Prérequis

Installez PyQt5 d'abord : pip install PyQt5

Qt Designer sera installé automatiquement avec.

📗 2. L'interface de Qt Designer

Quand vous ouvrez Qt Designer, vous voyez 4 zones principales :

🧰

Boîte à outils (Widget Box) — Contient tous les widgets disponibles (boutons, labels, zones de texte…)

🎨

Zone de conception (Form Editor) — Le canvas central où vous déposez et arrangez vos widgets

⚙️

Éditeur de propriétés (Property Editor) — Panel de droite avec toutes les propriétés du widget sélectionné

📁

Explorateur d'objets (Object Explorer) — Arborescence qui montre la hiérarchie de vos widgets

📙 3. Les types de fenêtres

Quand vous créez un nouveau projet dans Qt Designer (Fichier → Nouveau), trois types de fenêtre vous sont proposés :

🔲

Main Window — Fenêtre complète avec menu, barre d'outils et barre de statut. Pour les applications principales.

📦

Dialog — Petite fenêtre de dialogue (ex : confirmation, paramètres). Sans barre de menu.

🏷️

Widget — La forme la plus simple. Une fenêtre basique sans éléments supplémentaires.

💡 Quel type choisir ?

• Pour l'examen du Bac, le type Widget est le plus courant et le plus simple.

• Dialog est utilisé pour les fenêtres pop-up (connexion, alerte).

• Main Window est pour les applications avec menu (Fichier, Édition…).

📕 4. Les widgets (éléments d'interface)

Un widget est un élément visuel de votre interface. Qt Designer propose des dizaines de widgets dans la boîte à outils. Voici les plus importants :

📝 QLabel — Étiquette (texte non éditable)
🔘 QPushButton — Bouton cliquable
✏️ QLineEdit — Champ de texte sur une ligne
📄 QTextEdit — Zone de texte multi-lignes
☑️ QCheckBox — Case à cocher
⚪ QRadioButton — Bouton radio (choix unique)
📋 QComboBox — Liste déroulante
🎚️ QSlider — Curseur glissant
📊 QProgressBar — Barre de progression
📋 QListWidget — Liste d'éléments
🔗 QFrame — Cadre / conteneur visuel
🧱 QGroupBox — Groupe avec titre

1Comment ajouter un widget

1. Dans la Boîte à outils (à gauche), cliquez sur le widget souhaité.

2. Cliquez ensuite dans la zone de conception pour le placer.

3. Le widget apparaît dans votre fenêtre et dans l'Explorateur d'objets.

2Comment sélectionner un widget

• Cliquez dessus directement dans la zone de conception.

• Ou sélectionnez-le dans l'Explorateur d'objets (à droite).

Quand un widget est sélectionné, ses propriétés apparaissent dans l'Éditeur de propriétés.

3Comment déplacer et redimensionner

• Déplacer : glissez le widget avec la souris.

• Redimensionner : cliquez sur un widget puis tirez sur les poignées (carrés bleus) qui apparaissent aux coins et côtés.

4Comment supprimer un widget

Sélectionnez le widget et appuyez sur la touche Suppr (ou Delete) du clavier.

📘 5. Modifier les propriétés

Chaque widget a des propriétés que vous pouvez modifier dans le panel de droite (Éditeur de propriétés). Voici les plus utiles :

🏷️

objectName — Le nom unique du widget. Important pour l'identifier dans le code.

📝

text — Le texte affiché (pour les boutons, labels, champs…).

📐

geometry — Position (X, Y) et taille (Largeur, Hauteur) en pixels.

🔒

enabled — true/false : active ou désactive le widget.

👁️

visible — true/false : rend le widget visible ou invisible.

🅰️

font — Change la police, la taille et le style du texte.

💡 Bonnes pratiques pour les noms d'objets

Donnez des noms clairs et cohérents à vos widgets. Quelques conventions :

  • Bouton → pushButton_ok, pushButton_annuler
  • Champ texte → lineEdit_nom, lineEdit_motdepasse
  • Label → label_titre, label_erreur
  • Case à cocher → checkBox_majeur

⚠️ Important

Le champ objectName ne doit contenir aucun espace ni accent. Utilisez des underscores (_) pour séparer les mots.

Ce nom sera utilisé pour identifier le widget, alors choisissez quelque chose de logique !

📗 6. Les layouts (dispositions automatiques)

Les layouts permettent d'organiser automatiquement vos widgets. Sans layout, les widgets restent à une position fixe. Avec un layout, ils s'ajustent automatiquement quand la fenêtre change de taille.

Les layouts se trouvent dans la Boîte à outils, dans la section "Layouts" :

⬇️

Vertical Layout — Empile les widgets de haut en bas

➡️

Horizontal Layout — Aligne les widgets de gauche à droite

⊞

Grid Layout — Place les widgets dans une grille (lignes × colonnes)

📐

Form Layout — Disposition type label + champ (comme un formulaire)

1Appliquer un layout à la fenêtre

Faites un clic droit sur la fenêtre → choisissez "Lay Out" → puis le type de layout souhaité.

2Créer un layout et y mettre des widgets

1. Sélectionnez un widget (ou plusieurs) dans la zone de conception.

2. Cliquez sur le layout souhaité dans la Boîte à outils.

Les widgets sélectionnés seront automatiquement réorganisés.

3Layout imbriqués

Vous pouvez mettre un layout à l'intérieur d'un autre layout pour créer des interfaces complexes. Par exemple : un Vertical Layout principal, contenant deux Horizontal Layouts.

⚠️ Astuce

Pour "casser" un layout et repositionner les widgets librement, faites un clic droit → "Break Layout".

📙 7. Créer une interface complète — Pas à pas

Exemple : créer une fenêtre de connexion simple.

Étape 1 — Créer la fenêtre

Dans Qt Designer :

1. Fichier → Nouveau

2. Choisissez "Widget" puis cliquez sur "Créer"

3. Dans l'éditeur de propriétés, réglez :

  • windowTitle → Connexion
  • geometry → width → 400
  • geometry → height → 300

Étape 2 — Ajouter les widgets

Depuis la Boîte à outils, glissez-déposez ces éléments dans la fenêtre :

  • QLabel → pour le texte "Nom d'utilisateur"
  • QLineEdit → pour saisir le nom
  • QLabel → pour le texte "Mot de passe"
  • QLineEdit → pour saisir le mot de passe
  • QPushButton → "Se connecter"
  • QLabel → vide, pour afficher un message

Étape 3 — Organiser avec un layout

1. Sélectionnez tous les widgets de la fenêtre (Édition → Sélectionner Tout, ou clic + glisser).

2. Cliquez sur le bouton "Vertical Layout" dans la Boîte à outils.

Les widgets s'organisent automatiquement en colonne.

Étape 4 — Configurer les noms d'objets

Noms à donner à chaque widget (objectName) :

  • QLineEdit nom → lineEdit_nom
  • QLineEdit mot de passe → lineEdit_mdp
  • QPushButton → pushButton_connexion
  • QLabel message → label_message

Étape 5 — Modifier les textes affichés

Sélectionnez chaque widget et changez la propriété text dans l'éditeur de propriétés :

  • QLabel 1 → Nom d'utilisateur :
  • QLabel 2 → Mot de passe :
  • QPushButton → Se connecter
  • QLabel message → laissez vide pour l'instant

Étape 6 — Sauvegarder

Fichier → Enregistrer (ou Ctrl + S)

Sauvegardez votre fichier sous le nom interface_connexion.ui

C'est ce fichier .ui que vous pouvez donner à un développeur pour l'intégrer dans un programme.
