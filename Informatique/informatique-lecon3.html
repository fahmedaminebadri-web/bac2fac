📘 1. Qu'est-ce que Qt Designer ?

Qt Designer est un outil graphique inclus dans le framework Qt qui permet de concevoir visuellement des interfaces utilisateur (GUI) sans écrire de code. On glisse-dépose les éléments, on ajuste les propriétés, et Qt Designer génère un fichier .ui (format XML).

Au Baccalauréat tunisien, Qt Designer est utilisé avec PyQt5 (Python + Qt) pour créer des applications graphiques simples.

🎯

Objectif — Créer une interface graphique (fenêtre, boutons, champs de texte…)

📄

Sortie — Fichier .ui (convention XML) lisible par PyQt5

⚡

Avantage — Gain de temps énorme par rapport au codage manuel des interfaces

🐍

Avec Python — On utilise PyQt5 ou PyQt6 comme bibliothèque

⚠️ Installation (Tunsie — Bac Info)

Installez PyQt5 avec la commande : pip install PyQt5

Qt Designer se trouve dans le dossier d'installation de Python ou se lance via la commande : designer

📗 2. Interface de Qt Designer

Qt Designer s'organise en 4 zones principales :

🧰

Boîte à outils (Toolbox) — Liste des widgets disponibles (boutons, labels, zones de texte…)

🎨

Zone de conception — Canvas central où on dépose et arrange les widgets

⚙️

Propriétés (Property Editor) — Liste des attributs du widget sélectionné (taille, texte, couleur…)

📁

Explorateur d'objets — Arborescence hiérarchique des widgets de la fenêtre

Types de fenêtres

🔲

Main Window — Fenêtre complète avec menu, barre d'outils, zone centrale et barre de statut

📦

Dialog — Petite fenêtre de dialogue (sans barre de menu)

🏷️

Widget — Fenêtre basique, le type le plus simple

📙 3. Les widgets principaux

Un widget est un élément d'interface (contrôle graphique). Voici les widgets les plus utilisés au Bac :

📝 QLabel — Étiquette (texte non éditable)
🔘 QPushButton — Bouton cliquable
✏️ QLineEdit — Champ de texte sur une ligne
📄 QTextEdit — Zone de texte multi-lignes
☑️ QCheckBox — Case à cocher
⚪ QRadioButton — Bouton radio (choix unique)
📋 QComboBox — Liste déroulante
🎚️ QSlider — Curseur glissant
📊 QProgressBar — Barre de progression
📋 QListWidget — Liste d'éléments
🔗 QFrame — Cadre / conteneur visuel
🧱 QGroupBox — Groupe avec titre

Propriétés importantes

🔤

objectName — Nom unique pour identifier le widget dans le code

📝

text — Texte affiché (pour labels, boutons…)

📐

geometry — Position (x, y) et taille (width, height)

🔒

enabled — Active ou désactive le widget

📕 4. Créer une interface complète

Voici les étapes pour créer une fenêtre de connexion simple :

Étape 1 : Créer le projet

Dans Qt Designer :

1. Fichier → Nouveau → Widget

2. Définir la taille de la fenêtre (Property Editor → geometry → width: 400, height: 300)

3. Changer le titre (Property Editor → windowTitle → "Connexion")

Étape 2 : Ajouter les widgets

// Disposition de l'interface
┌──────────────────────────────────┐ │ 🎯 Connexion │ ├──────────────────────────────────┤ │ │ │ Nom d'utilisateur: │ │ [________________] │ │ │ │ Mot de passe: │ │ [________________] │ │ │ │ [Se connecter] │ │ │ └──────────────────────────────────┘

Étape 3 : Organiser avec des layouts

Les layouts gèrent automatiquement le positionnement et le redimensionnement des widgets :

⬇️

QVBoxLayout — Empilement vertical

➡️

QHBoxLayout — Alignement horizontal

⊞

QGridLayout — Grille (lignes × colonnes)

Étape 4 : Configurer les noms d'objets

Noms d'objets à définir (très important !) :

• lineEdit_nom — Champ nom d'utilisateur

• lineEdit_mdp — Champ mot de passe

• pushButton_connexion — Bouton de connexion

• label_message — Label pour afficher un message

Étape 5 : Sauvegarder

Enregistrer le fichier sous interface_connexion.ui (Fichier → Enregistrer)

📒 5. Connecter le .ui au code Python

Une fois l'interface créée, on l'utilise dans un script Python avec PyQt5.

Méthode 1 : Chargement direct du .ui

// fenetre.py — Chargement dynamique du .ui
import sys from PyQt5.QtWidgets import QApplication, QMainWindow from PyQt5.uic import loadUi class FenetreConnexion(QMainWindow): def __init__(self): super().__init__() loadUi("interface_connexion.ui", self) # Connecter le bouton self.pushButton_connexion.clicked.connect(self.verifier_connexion) def verifier_connexion(self): nom = self.lineEdit_nom.text() mdp = self.lineEdit_mdp.text() if nom == "admin" and mdp == "1234": self.label_message.setText("Connexion réussie !") else: self.label_message.setText("Erreur de connexion.") app = QApplication(sys.argv) window = FenetreConnexion() window.show() sys.exit(app.exec_())

Méthode 2 : Compilation du .ui en Python

Pour convertir le .ui en fichier Python (.py) :

Dans le terminal : pyuic5 -x interface_connexion.ui -o interface_connexion.py

Ensuite, dans votre code :

from interface_connexion import * # L'interface est directement accessible

⚠️ Signaux et slots

En PyQt5, les événements sont gérés par le système signal/slot :

• Un signal est émis quand quelque chose se produit (clic sur un bouton, changement de texte…)

• Un slot est une fonction qui répond à ce signal

Syntaxe : widget.signal.connect(slot_fonction)

📓 6. Projet final : Calculatrice

Construisons une calculatrice simple avec Qt Designer et PyQt5.

Interface (Qt Designer)

// Disposition de la calculatrice
┌──────────────────────────┐ │ [ 0 ] ← Display │ ├──────────────────────────┤ │ [7] [8] [9] [/] │ │ [4] [5] [6] [*] │ │ [1] [2] [3] [-] │ │ [C ] [0] [=] [+] │ └──────────────────────────┘

Code Python complet

// calculatrice.py
import sys from PyQt5.QtWidgets import QApplication from PyQt5.uic import loadUi class Calculatrice: def __init__(self): loadUi("calculatrice.ui", self) self.show() # Connecter tous les boutons numériques self.pushButton_0.clicked.connect(self.chiffre) self.pushButton_1.clicked.connect(self.chiffre) # ... (faire de même pour les autres) self.pushButton_egale.clicked.connect(self.calculer) self.pushButton_clear.clicked.connect(self.clear) def chiffre(self): texte = self.label_display.text() self.label_display.setText(texte + self.sender().text()) def calculer(self): try: resultat = eval(self.label_display.text()) self.label_display.setText(str(resultat)) except: self.label_display.setText("Erreur") def clear(self): self.label_display.setText("0") app = QApplication(sys.argv) calc = Calculatrice() sys.exit(app.exec_())

⚠️ Conseils pour l'examen du Bac

1. Créez d'abord l'interface dans Qt Designer et sauvegardez-la.

2. Notez tous les noms d'objets (objectName) avant de coder.

3. Testez l'interface puis le code séparément.

4. Utilisez la méthode loadUi pour plus de flexibilité.
